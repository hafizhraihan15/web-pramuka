ℹ️ Panduan Setup

Website ini menggunakan Supabase sebagai database dan Cloudinary untuk penyimpanan foto. Pastikan kamu sudah:

  1. Membuat project di supabase.com
  2. Menjalankan SQL di server/database.sql
  3. Mengisi SUPABASE_URL dan SUPABASE_ANON_KEY di js/app.js dan js/admin.js
  4. (Opsional) Daftar di cloudinary.com untuk upload foto
